# AI Agent Bot

## To run

### ENV Variable

`export OPENAI_API_KEY="...data"`

### Installation

`python -m venv venv`

`source ./venv/Scripts/activate`

`pip install -r requirements.txt`

### Running OpenAI agent

`py gpt.py`
